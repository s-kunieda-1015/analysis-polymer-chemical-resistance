# Output Directory

This directory contains all outputs generated by the analysis scripts.

## Subdirectories (Not tracked by Git)

### Main/
Main figures for the paper (Fig1, Fig3-9):
- Heatmaps, ROC curves, feature importance plots
- Chi parameter analysis figures
- Model comparison visualizations

### Sub/
Supplementary figures and analysis results:
- S1-S6 supplementary figures
- Model evaluation metrics (CSV files)
- Cross-validation results

### Model/
Trained machine learning models:
- `chemical_resistance_model.pkl` - GBDT model for resistance prediction
- `chi_parameter_model.pkl` - XGBoost model for Ï‡ parameter prediction
- `crystallinity_model.pkl` - RandomForest model for crystallinity classification

### LOGOCV/
Leave-One-Group-Out Cross-Validation results:
- ROC curve data for each polymer/solvent group
- Confusion matrix data
- Performance metrics by group

## Note

All files in this directory are generated by running `src/main_analysis.py`.
These outputs are excluded from version control as they can be reproduced from the source code and data.
